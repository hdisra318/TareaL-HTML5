Historia HTML



HTML es el lenguaje de marcado de hipertexto, que permite estructurar y organizar el texto para que sea
legible y permite establecer ciertas relaciones, hipervínculos y otras modificaciones por medio de etiquetas.
El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, trabajador del CERN (Organización
Europea para la Investigación Nuclear) propuso un nuevo sistema de "hipertexto" para compartir documentos.
En 1991, Tim Berners-Lee publicó un documento titulado "HTML Tags", conocido como el origen de HTML. En
1995 se publicó la segunda versión o HTML 2.0, que en realidad era una petición de estandarización. Es decir,
HTML 2.0 fue la primera propuesta oficial de estandarización de HTML. Desde entonces, HTML ha evolucionado
y ha pasado por varias versiones, cada una con nuevas características y mejoras.

Versiones Principales

  1. HTML : Tim Berners-Lee propuso un nuevo sistema de hypertexto en 1980 y junto a Robert Cailiau lo presentaron en una
    convocatoria de sistemas de hipertexto para internet y lo llamaron WorldWideWeb o W3, pero el primer articulo formal con su
    descripcion, llamado HTML Tags, fue publicado hasta 1981.
    En 1993 la IETF(Internet Engineering Task Force) propuso convertir HTML en un estandar y publicaron como propuestas HTML y HTML+, pero ninguna
    logro convertirse oficialmente en un estandar.
  2. HTML 2.0 : Fue el primer estandar oficial de HTML, organizado por la IETF en 1995
  3. HTML 3.2 : A partir de 1996 la World Wide Web Consortium o W3C se convierte en la encargada de punlicar los estandares de HTML.
    La version 3.2 de HTML fue publicada en Enero de 1997, entre sus mocedades se encontraban applets de Java y textos enmarcados alrededor de imagenes.
  4. HTML 4.0 : Publicada en Abril de 1998, esta version permite la incorporacion de archivos CSS y pequeños programas o scripts en el
    diseño de paginas web, entre otos avances que trajo este estandar se encuentran una significativa mejora en la accesibilidad de las páginas
    diseñadas, la posibilidad de elaborar tablas complejas e importantes mejoras en la confección y uso de formularios.
  5. HTML 5.2 : Debido a que la W3C se centro en el desarrollo de XHTML despues de lanzar la version anterior de HTML, en el 2004 Apple Mozilla y Opera
    se unieron y formaron el grupo Web Hypertext Application Technology Working Group o WHATWG, este grupo desarrollo un primer borrador de
    lo que seria HTML 5.0 y lo publico en 2008, debido a la presion generada por este grupo la W3C retomo el desarrollo de HTML y en 2014 presentaron el
    nuevo estandar HTML 5.2.
