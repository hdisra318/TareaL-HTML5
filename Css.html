Historia

En la década de los 90's las páginas web que existían no tenían un diseño atractivo para la vista, y estos eran complicados y poco flexibles de escribir, pues estaban integrados en el HTML, para lo cual el organismo W3C (World Wide Web Consortium), encargado de crear todos los estándares relacionados con la web, propuso la creación de un lenguaje de hojas de estilos específico para el lenguaje HTML. Las propuestas que se tuvieron en cuenta fueron la CHSS (Cascading HTML Style Sheets) y la SSP (Stream-based Style Sheet Proposal).

Más tarde entre finales de 1994 y 1995 se definió un nuevo lenguaje que tomaba lo mejor de cada una de las propuestas mencionadas y lo llamaron CSS (Cascading Style Sheets). Y a finales de 1996, el W3C publicó la primera recomendación oficial, conocida como "CSS nivel 1", donde proporcionaba una serie de propiedades y valores que ayudaban a definir estilos como colores, fuentes, márgenes y etc.

Y el primer navegador con soporte completo de CSS 1 fue la versión para Mac de Internet Explorer 5, que se publicó en el año 2000. Actualmente los desarrolladores ocupan CSS 3 donde pueden crear diseños y estilos más sofisticados y responsivos para desarrollar aplicaciones web completas y modernas.

Versiones de CSS

Versión Descripción
CSS1 Fue la primera versión que existió, surgió en el año 1996. Contiene todas las funcionalidades básicas como la sintaxis, los atributos o las propiedades de las fuentes. Lo más relevante fue el hecho de que todos los estilos se podían reunir en un documento externo al HTML, separando la estructura de lo visual.
CSS2 Apareció dos años después de la primera versión, en 1998, con el objetivo de seguir mejorando el lenguaje, pues para esta versión se le sumaron, de la versión de CSS1, propiedades como los posicionamientos absolutos y relativos, z-index, entre otros más; lo que provocó una revolución visual, pues ahora los desarrolladores podían colocar contenedores e imagenes donde ellos quisieran. Pero en 2011, debido a varios problemas de funcionalidad se agregaron nuvas especificaciones, dando lugar a CSS2.1
CSS3 Esta versión se empezó a desarrollar dese 1999 y es la que se usa actualmente, la cual se publicó en el año 2011 y hasta el día de hoy contiene las especificaciones más completas y amplias. Su gran cambio fue que esta versión se plantea como una suma de módulos, los cuales van sumando funcionalidades al lenguaje, lo cual permite que CSS3 evolucione. Una de las grandes aportaciones fue flexbox y CSS grid, los cuales resultaron una mejora en el posicionamiento gráfico de las cosas. Y otras innovaciones que trajo esta versión fueron los media queries, los cuales permitían el diseño responsive para poder adaptar de forma sencilla las aplicaciones web en los diferentes tipos de pantalla que existen en la actualidad.

Frameworks de CSS

A lo largo de los años han surgido varios Frameworks para CSS, los cuales son un conjunto de herramientas que nos permiten, en este caso, editar de una manera más rápida las hojas de estilo en cascada y crear temas de diseño más consistentes, como lo son:

  • Tailwind CSS
  • Bootstrap
  • Materialize CSS
  • Bulma
  • Foundation
  • Open Props
